Simple Setup

Up and running in minutes

No configuration files. No command-line wizardry. Just open, block, and focus.

Download the installer for your OS. Run the installer script and FocusKit launches instantly with a clean, native window. No dependencies required.

2
Add distraction sites

Type in any domain you want blocked — or pick a preset like Social Media, News, or Gaming. Sites are blocked at the system level, no browser extension needed.
